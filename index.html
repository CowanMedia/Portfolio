Playlist Application using Firebase / AJAX / Soundcloud API / MaterializeCSS

MIXED!

Firebase / AJAX / Soundcloud API / MaterializeCSS

https://cowanmedia.github.io/ProjectOne/

MIXED is a group project from the DU full stack web dev program. I was responsible for the frontend and used MaterializeCSS and Jquery. MIXED is a collaborative playlist application using the Soundcloud API. Similar to a jukebox, MIXED allows people to add songs to a group playlist. Imagine being at a party, and MIXED is like a jukebox - from the MIXED site you could create a new playlist for the party and everyone could add a song they want to hear. Same thing for restaurants and cafes. Instead of having to listen to what's playing, you can add a favorite song you want to hear!

In this demo of the MIXED app, you can:

  • check_circleCreate a new playlist and add songs
  • check_circleClick/play an existing playlist
  • check_circlePlay/delete songs
  • check_circleSearch the Soundcloud library by keyword to find new songs
  • check_circleAdd songs from the Soundcloud library to the playlist

This web app is in a early production stage... The idea would be to have a login page for an admin to "own" a playlist and have control over who can add/delete songs. There would also be a way to save the playlist and share it (like a mixed tape).
